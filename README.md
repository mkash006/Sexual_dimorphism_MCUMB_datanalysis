# Sexual_dimorphism_MCUMB_datanalysis
This repository contains data and [R code](annotated_Rscript.md) associated with a manuscript in review, "*Resource Limitation Shapes the Evolution of Sexual Dimorphism and Condition Dependence in Drosophila melanogaster*." This repository provides two different versions of the same dataset. a) [Untidy_dataset](main_dat.xlsx) - Untidy raw data set which contains redundancies noted down during data collection process.  b) [cdsd_data](cdsd_dat.csv) - Tidy version of the dataset, which you can use as an input file if you want to follow along with the analysis written in the annotated R script.
